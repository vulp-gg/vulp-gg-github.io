Getting Started

To get started, download our loader from here.

Make sure your antivirus is disabled before downloading.

It is highly recommended to run our loader from a USB drive.

It is best to download it straight to your USB drive.

Requirements

  • Windows 10 (Insider/preview builds and custom Windows ISOs are not supported)
  • 64-bit CPU with AVX instruction set
  • Fast boot disabled
  • No running hypervisors (some machines might require disabling virtualization)
  • Antivirus disabled
  • Administrator privilege
  • Secure boot disabled

How to Use the Loader

Make sure you have your game closed before launching the loader.

How to Inject

  1. Run loader as administrator from USB drive (doing otherwise is not recommended and might result in bans). (You might get a message that you have to turn something off, you can get more details below)
  2. Login or create an account if you do not have such already.
  3. Click and highlight the product you want to inject on the left side. You can see the status and expiration date on the right.
  4. Click "Inject" to load the product.
  5. Wait for the loader to fully close.
  6. Unplug your USB.
  7. Now you can start your game.

How to Activate a Key

  1. Run loader as administrator from USB drive (doing otherwise is not recommended and might result in bans). (You might get a message that you have to turn something off, you can get more details below)
  2. Login or create an account if you do not have such already.
  3. Paste in your key in the "License Key" field and press the "Activate Key" button. If you encounter any issues make sure you have no extra space in your key!

How to Disable Fast Boot

  1. RECOMMENDED: You can disable fast boot in your BIOS (you should check your motherboard manual for more details)
  2. Disable fast boot in Windows
    1. Open Control Panel
    2. Select Power Options
    3. On the left side, click on "Choose what the power button does"
    4. Click "Change settings that are currently unavailable"
    5. Make sure there is no checkmark next to "Turn on fast startup (recommended)"; if you do not see this option, you have it already disabled in BIOS
    6. Click "Save changes"
    7. Restart your computer

How to Disable Hypervisors

Official hypervisors (Hyper-V, etc):

  1. Open Command prompt as Administrator
  2. Run this command: bcdedit /set {current} hypervisorlaunchtype off
  3. Restart your computer

Unofficial hypervisors:

  • Disable virtualization in BIOS or get instructions from the maker of the hypervisor you are running

Common Issues

Error 0xc01c0013

  1. Make sure UAC (User Account Control) is enabled and file virtualization is enabled
  2. Download and run this registry file: here
  3. Restart your computer

If the above does not help you (does not work on Windows Home):

  1. Press the Windows + R keys together to open the Run dialog box and type secpol.msc. Then, it will open the Local Security Policy window.
  2. Expand Local Policies on the left-hand pane and click Security Options.
  3. Open 'User Account Control: Virtualize file and registry write failures to per-user locations' and make sure it is enabled.
  4. Restart your computer
  5. Make sure you are not running unofficial Windows ISOs

Error 0xc00000bb

Your Windows version is not supported. Make sure you are running official Windows and you are not on Insider/Preview builds. You can download Windows installation media from Microsoft website.

Error 0xc0000365 / 0xc0000906 / 0xc0000907 / 0xc000009a / 0xc0000022

  1. Uninstall FaceIT AC/Vanguard/ESEA anti-cheat or any other anti-cheat which starts up on boot. Make sure to restart your computer afterwards.
  2. Disable your antivirus.
  3. If none of the above work, reinstall Windows using official Windows ISO.
